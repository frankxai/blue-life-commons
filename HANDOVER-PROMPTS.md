# Blue Life Commons — Handover Prompt Library

Ready-to-use, high-fidelity prompts for other harnesses (Codex / GPT image, Antigravity / Nano Banana / Gemini grounded, Higgsfield, etc.).

**When to hand over:**
- Heavy text, exact labels, charts, diagrams, complex infographics (Grok Imagine is artistic but text can be unreliable — see imagine skill).
- Grounded factual diagrams where web/research grounding in the target model helps.
- High-volume or specific model strengths (e.g. NB2 excellent text + research per acos-visual-gen).

**How to use:**
- Copy the full prompt block.
- Add any additional facts or constraints from latest species/region pages.
- For Antigravity/Infogenius: prepend research facts + specify style from their system (standard, photorealistic, minimalist, technical).
- For Codex: emphasize "perfect legible typography, no spelling errors, integrated graphic text".

## Infographic Example (Ethical Observation)
```
Clean modern infographic: "Ethical Whale & Dolphin Observation — Blue Life Commons"

Layout: vertical or 16:9 with clear visual hierarchy.

Top: title + small icon lockup.

Sections (large legible text, integrated):
1. Keep distance — minimum approach distances (boat 100m, swim 30m or local rules). Icon of boat and animal with space.
2. No chasing or encircling. Calm parallel approach only.
3. Limit time — 30 min max per group, one boat at a time in sensitive areas.
4. No touching, feeding, or flash photography.
5. Report & record responsibly — use apps, note behavior, contribute to commons.

Visual style: premium educational infographic. Ocean-inspired palette (deep navy base, teal accents, gold highlights). Clean sans typography, high contrast, large readable text. Subtle wave or current motifs as dividers. Minimal icons that feel scientific yet warm.

Audience: citizen scientists and ecotourism operators. Professional, empowering, not preachy.

Aspect: 9:16 or 16:9
Text must be perfectly legible. Use generous sizing and high contrast.
```

## General Handover Tips by Target

**Codex / GPT-image family:**
- Add: "perfectly legible high-quality typography, spelling accurate, no text artifacts, text is part of the graphic design".
- "Vector clean illustration style for logos and diagrams".

**Antigravity / NanoBanana / Gemini (Infogenius style):**
- Prepend 3-5 verified facts from the species/region md.
- Specify style: "photorealistic cinematic", "standard clean scientific illustration", "minimalist".
- Request "enable_grounding" or "research-grounded" if available.
- "High thinking level for accuracy".

**For any:**
- Always include palette hexes.
- Specify "warm hopeful lighting with golden surface rays".
- Mention "Blue Life Commons visual system" for consistency.
- End with "professional premium quality, no visual artifacts, emotionally resonant".
