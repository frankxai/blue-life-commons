# Mission Blue — Hope Spots Integration Guide

| Field | Value |
|---|---|
| Partner | Mission Blue (missionblue.org) |
| Programme | Hope Spots |
| Coverage | 245 Hope Spots · ~80 million km² |
| BLC Relevance | Region anchor citations · MPA boundary overlap via Protected Planet |
| Contact | missionblue.org/contact |

Mission Blue maintains the global Hope Spots programme — a network of special places in the ocean that Sylvia Earle and her team believe are critical to the health of the sea. With 245 designated Hope Spots covering approximately 80 million km², this network represents one of the most authoritative geographic reference frameworks available for marine region work. BLC integrates Hope Spots as compelling, recognised anchors for region briefings, guardian context, and public-facing content.

---

## Integration Points

### Region Files (`content/regions/`)

BLC region briefings reference Hope Spot designations where applicable. When a region overlaps with a Hope Spot, the briefing should include a link to the Hope Spots database at [missionblue.org/hope-spots](https://missionblue.org/hope-spots) and note the designation date and significance. This provides readers with direct access to Mission Blue's authoritative documentation for that area.

### OIS Protected Planet Connector

The Ocean Intelligence System `protected-planet` connector provides MPA boundary data that partially overlaps with Hope Spots. Hope Spots are not always formal MPAs — many are aspirational designations — so the overlap is partial and should be noted when it exists. When the OIS returns MPA polygons for a region, cross-reference against Hope Spot records where available to surface both protection tiers in guardian briefings.

### Partnership and Data Exchange

Data exchange is facilitated via [ProtectedSeas Navigator](https://navigator.protectedseas.net/), which indexes 20,000+ MPAs and includes annotations relevant to Hope Spot areas. For direct collaboration, contact Mission Blue through their website. A formal data partnership would enable more precise Hope Spot polygon data in OIS region queries.

### For Creators

Hope Spots offer strong narrative anchors for BLC content. When writing region-specific wisdom articles, dive guides, or field mission briefs, citing the Hope Spot designation grounds the content in a globally recognised conservation framework. The name recognition of Mission Blue and Sylvia Earle carries significant public reach.

### For NGOs Using Guardian Briefings

Guardian briefings generated for Hope Spot regions can feed directly into Mission Blue's monitoring and advocacy work. If an NGO is operating within a Hope Spot, the BLC region context — species records, threat indicators, field observations — provides structured intelligence that complements Mission Blue's narrative reporting.

---

## Attribution Standard

All BLC content that references Hope Spots cites Mission Blue as an authoritative partner:

```
Hope Spot designation: Mission Blue (missionblue.org/hope-spots)
```

Where a specific Hope Spot record is cited, include the Hope Spot name and designation year alongside the link.
