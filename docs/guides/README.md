# How to Use This — Audience Guides

The Ocean Intelligence System serves real people doing real work for real animals. This is the index of role-specific guides: find yourself below, and follow the one written for you.

New here? Start with [**GETTING-STARTED.md**](GETTING-STARTED.md) — the whole system in one read: the three-layer triad, a 5-minute tour, how to install `marine-mcp`, and how to make your first contribution (Git *and* no-Git paths).

## The rules everyone shares

No matter which guide you read, these standards apply — they are never subject to community vote:

- **Every factual claim needs a source** ([`SOURCES.md`](../../SOURCES.md)).
- **Animal welfare over content value** ([`ETHICS.md`](../../ETHICS.md)) — and welfare is a first-class interest the system represents, not just a constraint ([`WELFARE.md`](../../WELFARE.md)).
- **No precise locations of vulnerable or individual animals** — ever, including post-release.
- **The system assembles evidence; it never adjudicates** clinical or legal verdicts.
- **No anthropomorphic claims as fact.**
