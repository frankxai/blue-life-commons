# brand/

Blue Life Commons visual brand system and generated assets.

## Structure
- `logos/` — future master SVGs + lockup concepts
- `tokens/` — brand-tokens.css (source of truth for color + basic UI)
- `visuals/` — all generated key art, logos, infographics, patterns (from Grok + handovers)
- `prompts/` — full prompt archives per batch with metadata and learnings

## Core Documents (at repo root level)
- `VISUAL-STRATEGY-PLAN.md` — the sophisticated roadmap
- `VISUAL-ASSETS-REGISTRY.md` — every asset with prompt, grounding, QA
- `BRAND-GUIDELINES.md` — usage rules
- `HANDOVER-PROMPTS.md` — ready prompts for Codex, Antigravity, Infogenius, etc.

## Generation Notes
All assets created with:
- Locked palette from .github/hero.svg + warm life accents
- Grounded facts from content/ species & regions
- Warm, hopeful, reverent, professional tone
- Experiments on text, composition, icon vs full lockup

See the plan and registry for details and status.
