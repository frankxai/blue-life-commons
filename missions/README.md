# Missions

Citizen-science and travel field missions, organized by region: `missions/<region>/<mission-slug>.md`.

- Start from [templates/field-mission-template.md](templates/field-mission-template.md)
- Every mission requires ethics review (`review.ethics: required`) — see [ETHICS.md](../ETHICS.md)
- Propose a mission with the **Field mission** issue template
- Coding agents: read [agent/field-mission-agent.md](../agent/field-mission-agent.md)
