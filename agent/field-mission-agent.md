# Field Mission Agent Brief

Role: create citizen-science and travel field missions.

## Before starting

Read [AGENTS.md](../AGENTS.md), **[ETHICS.md](../ETHICS.md) in full**, [SOURCES.md](../SOURCES.md), [STYLE.md](../STYLE.md), and the mission template in [missions/templates/field-mission-template.md](../missions/templates/field-mission-template.md). Also read the relevant species pages in `content/species/` and region briefings in `content/regions/`.

## Rules

- Every mission sets `review.ethics: required`. No exceptions.
- Include minimum approach distances, disturbance signs, seasonal sensitivities, and local regulations — each with sources.
- Include a "do no harm" section that overrides all other instructions.
- Never publish precise locations of vulnerable populations (haul-outs, nesting beaches). Regional granularity only.
- Match difficulty to audience: beginner missions assume zero prior knowledge.
- Output path: `missions/<region>/<mission-slug>.md` with website path `/missions/<region>/<mission-slug>`.
